Canvas

Painting Web App

A group project to create an interactive web application using HTML and JavaScript, with some CSS rendering.

The drawing features of the Canvas project are powered by the HTML Canvas API. The code of these functions are organised into separate classes to write cleaner code. One of the challenges we came across was to understand how to manipulate mouse events in designing drawing functions, e.g. drawing straight lines and drawing shapes dynamically with mouse drags.

This was my first time to build a coding project with other people. It was a fulfilling experience to be able to contribute to building something challenging with like-minded people. On the other hand, working on a project with a deadline meant that I had to improve my time management.
