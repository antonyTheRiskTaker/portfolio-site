Quiz App

Learn by Testing Yourself

An Express application that performs server-side rendering to allow the user to quiz themselves.

Building a web server offers challenges beyond making simple web app using HTML, CSS and JavaScript. It tests your understanding of how an API handles requests and sends back response. The skill of designing a basic SQL database is also essential to ensuring that data is safely stored and can be fetched and updated as required. Personally, I find the time when I was researching on various parts of the quiz game logic (e.g. randomly select a collection of CS questions for each game session) particularly satisfying.

Team work means that you need to be able to collaborate with people from all walks of life. Some people turn out to pleasant to work with, while others would require you to be more tactful so that a toxic relationship would not derail the whole project. Moreover, in the midst of a major lockdown, we also had our first taste of remote work.
