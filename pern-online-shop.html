PERN Online Shop

Bootcamp Capstone Project

A full-stack online shop using the React library.

While creating web pages is streamlined and made enjoyable by organising frontend code into self-contained componenets, it did take me while to learn how to manage states using Redux, with some help from my teammates and instructors.

It is easy to get over-ambitious when you have learnt a considerable amount of tools for web development and are eager to apply them all. It took some persuasion before I scrapped some of the features in order to deliver on time. The biggest takeaway, in my opinion, is to acknowledge where you fall short and ask for help from others if necessary.
